## I have created a sample registration program using Python.
## Mini “Bar Registration + Entry System” 😎

### What does the program do?
- Shows a welcome message
- Takes user name input and validates it (only alphabets allowed)
- Checks if all required fields are filled properly
- Takes mobile number input and validates it
- Takes age input and applies restriction (minimum age required: 18)
- Generates a unique access ID
- Shows final confirmation message after successful registration
